# My 3D Site

This project is a simple web application that displays a 3D model of a Russian doll (Matryoshka) in blue color. The model can be rotated 360 degrees to the left and right.

## Project Structure

```
my-3d-site
├── index.html       # Main HTML document
├── css
│   └── styles.css   # Styles for the web page
├── js
│   └── script.js     # JavaScript for loading and rotating the 3D model
└── README.md        # Project documentation
```

## Getting Started

To set up and run the project, follow these steps:

1. **Clone the repository** (if applicable):
   ```
   git clone <repository-url>
   cd my-3d-site
   ```

2. **Open `index.html`** in your web browser. You can simply double-click the file or use a local server.

3. **Dependencies**: Ensure you have a modern web browser that supports WebGL for 3D rendering.

## Features

- Displays a 3D model of a blue Russian doll.
- Allows rotation of the model 360 degrees using mouse or touch input.
